Vim-Block-Paste
====

You can insert visual-block to anywhere!

## Demo

## Requirement

- NeoVim >= 0.4

## Install

You can use the plugin manager. e.g. dein.vim
```vim
[[plugins]]
repo = 'Rasukarusan/vim-block-paste'
```

## Usage

Select blockwise by `C-v` and execute `:Block`

```vim
:'<,'>Block
```
